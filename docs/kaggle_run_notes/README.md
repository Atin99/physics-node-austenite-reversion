# Kaggle Run Notes

This folder summarizes the four archived training runs that were available at handoff.

Files:

1. `00_run_comparison_summary.md`
   Single-page comparison of all four runs and how to use them.
2. `01_diagnostic_50_epoch_run.md`
   Early diagnostic run kept for comparison only.
3. `02_long_150_epoch_swag_run.md`
   Long run with full-scale data coverage and SWAG checkpointing.
4. `03_fast_smoke_30_epoch_run.md`
   Minimal smoke-test run used to validate the fast training path.
5. `04_accelerated_60_epoch_run.md`
   Best time-bounded completed run from the final sequence.
